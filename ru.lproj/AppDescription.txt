1Password — это ваш личный помощник, обеспечивающий сохранение паролей и учетных данных. Это приложение делает простое хранение паролей реальностью, так как работает непосредственно с браузером, сохраняет имена пользователя и пароли и создает стойкий уникальный пароль для каждого сайта. Затем 1Password выполняет автоматическую и безопасную процедуру входа в систему, ввод данных кредитных карт и заполнение форм регистрации, шифруя информацию одним удобным для запоминания паролем. 

ПРИМЕЧАНИЕ. Для существующих пользователей 1Password 3.8. Для достижения оптимальных результатов удалите старую копию 1Password перед установкой версии из Mac App Store

И это еще не все! 1Password помогает легко решать следующие задачи: 

- использование нескольких имен пользователя на одном сайте;
- создание нескольких наборов учетных данных для заполнения регистрационных форм, опросов и корзин;
- безопасная беспроводная синхронизация данных между компьютерами Mac, телефонами iPhone и планшетами iPad через сервис Dropbox или локально через Wi-Fi;
- хранение конфиденциальной информации в виде безопасных заметок;
- сохранение лицензий на ПО и квитанций;
- упорядочение всех записей с помощью тегов и папок.

Новая версия 1Password переработана для использования уникальных возможностей OS X Lion, в том числе: 

- полноэкранный режим;
- более эффективная система безопасности, совместимая с Lion Sandbox;
- дополнительные улучшения системы безопасности с использованием калибровки PBKDF2;
- поддержка 64-разрядных систем для ускорения работы.

Другие возможности новой версии: 
- **Новинка** быстрый доступ к данным из строки меню;
- улучшенная поддержка браузерных расширений.


Доступны мобильные версии 1Password для iPhone, iPod Touch и iPad. Синхронизация и обновление с помощью Wi-Fi или через Интернет с использованием сервиса Dropbox для гарантии постоянного доступа к данным.
